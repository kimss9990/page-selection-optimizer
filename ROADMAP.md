# Smart Paper Fit - 개발 로드맵

## 완료된 기능
- [x] NFP (Minkowski Sum) 기반 정밀 네스팅
- [x] 알고리즘 선택 UI (빠른분석 / NFP / NFP+GA)
- [x] 유전 알고리즘 (GA) 최적화

## 다음 개발 계획

### 1. 성능 최적화
- GA 속도 개선
- NFP 캐싱 강화
- Web Worker 병렬 처리 개선

### 2. 회전 세분화
- 45도, 30도, 15도 등 더 많은 회전 각도 지원
- 사용자 설정 가능한 회전 스텝

### 3. 다중 도면 지원
- 서로 다른 도면을 한 종이에 혼합 배치
- 각 도면별 수량 지정

### 4. 결과 내보내기
- PDF 내보내기
- SVG 내보내기
- 인쇄용 레이아웃

### 5. 시각화 개선
- 효율 비교 그래프
- 배치 애니메이션
- 여백 시각화

### 6. 코드 정리
- 리팩토링
- 테스트 코드 추가
- 타입 안정성 강화
